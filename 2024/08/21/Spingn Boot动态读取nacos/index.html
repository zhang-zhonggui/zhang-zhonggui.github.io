2024/8/21 Spring Boot，Nacos

Spring Boot动态读取nacos配置文件

1、默认当前已经可以读取配置nacos配置文件修改成动态读取在获取参数类上添加@RefreshScope注解即可

@RequestMapping("/test")
@RestController
@RefreshScope
public class TestController {

    @Value("${zzg.zzg}")
    private String zzg;

    @GetMapping("/get")
    @ResponseBody
    public String get() {
        return zzg;
    }
}
